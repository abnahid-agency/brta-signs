# **App Name**: Kalash Driving Test

## Core Features:

- MCQ Practice: Practice multiple-choice questions in Bangla with instant answer validation and explanations. 
- Mock Exam Generation: Generate mock exams with a set number of random questions, a countdown timer, and pass/fail results.
- Traffic Sign Information: Generates and provides detailed information about traffic signs, including name (Bangla and English), category, and explanation based on image inputs. The tool can leverage LLMs for question generation about the specific behavior around a sign.
- User Authentication and Profiles: Allow users to register/login via Email or Google, edit profile information, and follow other users.
- Exam History Tracking: Track user exam history, including scores, mistakes, and time taken, and provide insights into strengths and weaknesses.
- Admin Content Management: Enable admin to create, edit, and delete topics, upload traffic signs, add MCQs, categorize signs, and manage users.
- Progress Tracking: Visually display user progress with total exams taken, best score, weak topics, and time spent practicing using charts.

## Style Guidelines:

- Primary color: Deep red (#B71C1C), evoking a sense of urgency and importance related to traffic laws and safety.
- Background color: Light desaturated red (#F2D4D4) creating a soft, non-distracting backdrop.
- Accent color: Yellow-gold (#FFAB40) to highlight calls to action and important elements, referencing traffic signals for caution and attention.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and short amounts of body text, providing a modern, tech-oriented feel, paired with 'Inter' (sans-serif) for longer body text to ensure readability.
- Use clear and recognizable icons, consistent with traffic signs and symbols, to aid navigation and comprehension.
- Implement a mobile-first, responsive layout with a clean, exam-like interface to reduce visual clutter.
- Use subtle micro-interactions and hover states to provide feedback and enhance the user experience, such as smooth paging transitions and progress bar updates.
